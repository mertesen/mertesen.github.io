#Distributed Computing Systems

**a.	Cluster Computing**
Daha fazla işlem gücü elde etmek için high speed local area network’te haberleşen homojen bilgisayar kümelerinin çalışmasıdır. Nodelardan biri master node ise yani hiyerarşi varsa beowulf cluster computing’dir. 

**b.	Grid Computing**
Sanal organizasyonların olmasıdır. Hetorojen Nodelardan oluşan ve genelde spesifik bilimsel projelerde kullanılan dağıtık sistem. Örn; Hadron çarpıştırıcısında kullanılmış

**c.	Cloud Computing**
Grid Computingin evrimleşmesinin sonucu olarak Cloud Computing çıkmıştır. Ticari olarak istenen/sipariş edilen miktarda  cpunun verilmesidir. Herkes için.
